We accept returns only if the item was damaged during delivery, and the return request must be submitted within 7 days of receiving your order. To initiate a return, please email us at info@ametsuchi.co with the following information:

 

- Your order number in the subject line

- A brief explanation of the issue

 

If your return is approved, we will provide a prepaid return shipping label along with detailed instructions on how to return the item. Returns sent without prior authorization will not be accepted. To be eligible, returned items must be:

 

- In the same condition as received (unworn and unused)

- Accompanied by proof of purchase
